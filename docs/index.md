
## 实验介绍
具体来说整个实验分为五个小实验：

- 环境配置与测试用例编写：配置实验环境，学习 SysY 语法。
- 词法分析与语法分析： 实现词法分析和语法分析，将源代码转化为语法树。
- 语义分析：实现符号表，基于语法树进行语义分析。
- 中间代码生成：把分析后的语法树转化为实验定义的中间代码。
- 目标代码生成：将中间代码转化为 RISC-V 64 汇编代码。
